Checklist de Entrega

Marque cada etapa conforme for concluída. O progresso é salvo automaticamente no seu navegador.

Antes de começar
  • Cliente criou a conta no Google Analytics e forneceu o ID (G-...)
  • Domínio registrado e DNS acessível
  • Briefing preenchido: nicho, público, objetivo, CTA principal
  • Textos, logo e imagens recebidos do cliente
Durante o desenvolvimento
  • Código GA4 inserido com o ID do cliente
  • Eventos de rastreamento configurados (CTA, formulário, WhatsApp)
  • Versão mobile testada (320px, 375px, 768px)
  • Velocidade verificada no Google PageSpeed Insights
  • Meta tags configuradas: title, description, og:image, canonical
  • robots.txt e sitemap.xml criados
Na entrega
  • Deploy feito e URL de produção funcionando
  • Cliente verificou o domínio no Google Search Console
  • Sitemap enviado no Search Console (/sitemap.xml)
  • GA4 testado em Tempo Real (evento aparece ao acessar a página)
  • Formulário / WhatsApp testado: mensagem chega corretamente
  • Acesso de Editor no GA4 configurado (se incluso no pacote)
Documentação entregue ao cliente
  • URL de produção enviada ao cliente
  • Acesso ao repositório entregue (se incluído no pacote)
  • Instruções básicas de uso / alteração de textos enviadas
  • Prazo de garantia e contato de suporte esclarecidos
